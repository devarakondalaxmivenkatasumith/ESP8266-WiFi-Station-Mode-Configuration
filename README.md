# ESP8266 WiFi Station Mode Configuration

This project demonstrates how to connect a NodeMCU ESP8266 to a WiFi network using Station Mode.

## Hardware Used
- NodeMCU ESP8266
- USB Cable
- WiFi Router

## Features
- Connects to WiFi network
- Displays IP Address
- Displays MAC Address
- Serial Monitor Output

## Arduino Library
- ESP8266WiFi.h
